# KArt Everything – Portfolio Website

KArt Everything is a personal portfolio website created to showcase my creative work, experiments, and web development projects. It serves as a central space where I present my growth as a developer and designer, highlighting both completed work and ongoing learning experiences.

---

## 🚀 My Journey in Web Development

This project marks the beginning of my journey into web development, starting with the core foundations of the web:

- **HTML** – structuring web pages and understanding how content is built  
- **CSS** – styling, layouts, and visual design  
- **JavaScript** – adding interactivity and dynamic behavior  

What started as curiosity about how websites work has grown into a hands-on learning process of building real projects, solving problems, and improving step by step through practice.
