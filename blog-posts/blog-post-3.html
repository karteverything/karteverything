Supabase Integration

Published: [21-10-2025]

Last Edited: Nov 3, 2025

My portraiture gallery comes with features that enables me to upload images, organize them, and ensuring the database reflects the newly uploaded images. I choosed Supabase because it makes authentication, storage, and database management much simpler to use.

Setting Up Supabase

First, I created a Supabase project that acts as a container, and set up my database. My table for storing images looks like this:

Data Type Description
id uuid Primary key, auto-generated
image_url text Supabase Storage URL
title text Name or title of the image
created_at timestamp When the image was uploaded

This simple database schema is enough to store all the relevant details about each portrait.

Building the Admin Page

  1. Login Authentication:

    2. Using Supabase Auth, I created a secure login page so only I can access the admin interface.

  2. Image Upload Form:

    3. A simple form that allows me to select an image, enter a title and description, and submit it. On submission, the image gets uploaded to Supabase Storage.

  3. Image Metadata Scripting:

    4. Once uploaded, a script automatically extracts metadata and a thumbnail appears.

  4. Database Insertion:

    5. Once the image is uploaded, I insert a record in the Supabase database with new image url and metadata.

  5. Cancel, Delete, Edit Options:

    6. Before confirming the upload, I can cancel it. After upload, I can delete an image or edit its title directly from the admin dashboard, and changes immediately reflect in the database.

Displaying the Gallery

Once the images are in the database, displaying them is straightforward. I query the database for all images and map them to my gallery layout, then I render them with a simple grid in my frontend.

Conclusion

Integrating Supabase has simplified my workflow. Now, instead of having to modify my static html source code - I can now simply add new portraits to my gallery just by logging in, uploading an image, and submitting the form. The combination of secure authentincation, structured database storage, and hosted images makes Supabase a perfect fit for my portraiture gallery.
